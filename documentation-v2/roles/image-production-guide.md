# 图片生产指南 V2 / Image Production Guide

> **能力定位**：统一负责图片提示词消费、批量生成、返工修复三件事。

---

## 一、输入

1. 当前主题 framework
2. 当前 task packet
3. 当前 prompt / image draft / review notes

---

## 二、输出

1. 生成报告
2. 图片草案或替换提案
3. 失败与返工清单

---

## 三、把生成和修复看成同一能力

### 初次生成

- 根据 prompt 批量出图

### 修复返工

- 根据 review note 或参考图重做

它们本质上都是“围绕统一视觉规范生产可用图片”。

---

## 四、工作流

1. 读取 task packet
2. 处理一个模块或一个小批次
3. 输出报告
4. 停止等待审核
5. 若退回，进入修复模式

---

## 五、不要做的事

- 不要一次性生成所有模块
- 不要私自改主题框架
- 不要在没有 review 的情况下批量覆盖已存在资产
