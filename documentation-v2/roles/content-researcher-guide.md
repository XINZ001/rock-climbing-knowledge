# 内容研究员指南 V2 / Content Researcher Guide

> **能力定位**：负责搜集文本类内容并输出结构化 draft。适用于知识库补充、训练卡研究、名人堂档案研究。

---

## 一、输入

执行前只需要 3 类输入：

1. 当前主题 framework
2. 当前 task packet
3. 当前已存在的 registry / draft 文件

---

## 二、输出

默认只输出：

1. `draft registry`
2. `batch report`

不直接修改前端运行时数据。

---

## 三、通用流程

1. 读取 framework，理解最小内容单位
2. 读取 task packet，限定本轮范围
3. 搜索并筛选来源
4. 结构化写入 draft
5. 输出本轮报告
6. 停止并等待 review

---

## 四、适用主题举例

### 知识库

- 搜延伸阅读
- 搜专家观点

### 训练手册

- 搜训练卡
- 搜协议参数

### 名人堂

- 搜 profile
- 搜采访
- 搜训练理念

---

## 五、通用质量门槛

1. 来源可信
2. 不猜测
3. 结构化字段完整
4. 明确标出缺口
5. 允许产出 draft，不强行产出最终结论
