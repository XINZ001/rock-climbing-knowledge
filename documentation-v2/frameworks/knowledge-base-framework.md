# 知识库框架 V2 / Knowledge Base Framework

## 一、主题定位

知识库回答的是：

- 这是什么
- 为什么重要
- 怎么理解

而不是：

- 今天怎么练
- 某个人是谁

---

## 二、最小内容单位

`knowledge point`

每个知识点是前端内容、视频、插图、延伸阅读的聚合中心。

---

## 三、每个知识点建议结构

1. 标题
2. 正文
3. 术语
4. 交叉引用
5. 插图
6. 视频
7. 延伸阅读
8. 专家补充

---

## 四、边界

### 属于知识库

- 技术
- 原理
- 规则
- 装备概念
- 风险认知

### 不属于知识库

- 可直接执行的训练菜单
- 人物生平和采访档案

---

## 五、与其他主题的关系

- 训练手册通过 `relatedKps` 反向引用知识库
- 名人堂可以通过 `relatedKps` 和 `relatedTopics` 指向知识库
