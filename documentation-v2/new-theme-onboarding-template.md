## 五、接入输出

接入完成后至少应产出：

1. 一个 framework
2. 一个 registry schema
3. 一个 Team Lead 任务包
4. 一个前端整合任务包

---

## 六、使用原则

如果你填完这份模板后发现：

- 最小内容单位和旧主题相同
- 输出结构和旧主题相近
- 调用能力几乎一样

那大概率不该新建一套 bot，只该新增 framework 和 task packet。
