# The Training Bible: Building a Better Climber

**Author:** Neil Gresham
**Source:** Climbing Magazine / Rock and Ice Magazine
**Format:** 8-phase, 12-month periodized training plan

---

## Series Overview

A complete year-long training plan based on periodization principles. Each phase is 6 weeks. Phases build upon each other — do not skip phases if possible.

| Phase | Title | Focus |
|-------|-------|-------|
| [Phase 1](phase-01-conditioning.md) | General Conditioning | Base fitness, aerobic capacity, flexibility |
| [Phase 2](phase-02-low-intensity-endurance.md) | Low-Intensity Endurance | High-volume climbing, route mileage |
| [Phase 3](phase-03-strength.md) | Strength Training | Bouldering intensity, hangboard, campus |
| [Phase 4](phase-04-power-endurance.md) | Power Endurance | 15–40 move sustained sequences |
| [Phase 5](phase-05-strength-power.md) | Strength and Power | Second strength cycle, refined hangboard |
| [Phase 6](phase-06-endurance.md) | Endurance | Return to low-intensity endurance, recovery |
| [Phase 7](phase-07-power-endurance.md) | Power Endurance | Second PE cycle, redpointing focus |
| [Phase 8](phase-08-peaking.md) | Peaking | Onsight/redpoint skills, performance climbing |

## Key Principles

- Warm up and cool down every session
- Listen to your body; adjust workload if not recovering
- Climb outside as much as possible — tie crag sessions to training goals
- For trips longer than 5 days: take 3 rest days before and after
- Between phases: take 4 rest days before starting the next phase
