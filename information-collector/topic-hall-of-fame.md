# 主题范围：攀岩名人堂 / Topic Scope: Climbing Hall of Fame

> **用途**：本文件定义"攀岩名人堂"这一主题的收录范围与分类框架，供信息收集员在搜索时判断目标人物是否在范围内，以及用什么角度切入搜索。
>
> **不含内容**：具体的搜索方法、页面获取方式、输出格式 → 见 `information-collector/information-collector-guide.md`
>
> **不含内容**：每位运动员的结构化信息收集清单、JSON 模板 → 见 `athlete-researcher/athlete-research-guide.md`

---

## 一、什么是"攀岩名人堂"

攀岩名人堂收录的是**在攀岩运动历史上留下了清晰印记的人**。这个印记可以是：

- 打破了攀岩的难度边界（首登极难线路、抱石问题）
- 赢得了竞技攀岩的最高荣誉
- 推动了攀岩的训练方法进化
- 定义了攀岩的某种风格或流派
- 在中国用户中具有重要的文化影响力

**核心判断标准**：这个人的名字，在攀岩圈里是否代表了某种值得了解的东西？

---

## 二、分类框架 / Category Framework

前端使用四个**贡献类型**类别，取代原来按时代或地域划分的方式。中国运动员不再单独设类，而是在各类别中平等参与，通过 `isChineseRepresentative` 字段标记，前端据此提供独立的跨类别筛选入口。

```
攀岩名人堂（前端 category 值）
│
├── elite（竞技精英）
│   在国际赛场改写竞技上限的运动员，无论国籍
│   ├── 先锋攀 / 难度攀岩（Lead / Sport）
│   ├── 抱石（Bouldering）
│   └── 速度攀岩（Speed）
│
├── explorer（探险攀登）
│   把户外技术攀登推入新境界的人
│   ├── 自由攀登先驱（Free Climbing Pioneers）
│   ├── 大岩壁与探险攀登（Big Wall / Alpine）
│   └── 无保护攀登（Free Solo）
│
├── legend（历史先驱）
│   奠定现代攀岩底层逻辑的开创性人物
│   ├── 动作语言与抱石文化的定义者
│   └── 早期自由攀登革命的引领者
│
└── innovator（训练革新）
    发明或系统化了影响整个攀岩训练界的方法与工具
    ├── 训练工具发明者（campus board、Moon Board 等）
    └── 运动科学引入者（系统性训练体系建立者）
```

**中国运动员处理规则**：
- 根据贡献类型归入以上四个类别（不另设独立 category）
- 在 `athlete-registry.json` 中标记 `"isChineseRepresentative": true`
- 前端提供"中国运动员"跨类别筛选按钮
- **理由**：中国世界冠军与国际顶级选手并列同一类别，准确反映其实际成就量级；文化认同通过筛选器而非降级分类来满足

**原 A/B/C/D/E → 新类别映射**：

| 原分类 | 新 category |
|--------|------------|
| A. 竞技攀岩 | `elite` |
| B. 户外技术攀登 | `explorer` |
| C1. 训练方法创始人 | `innovator` |
| C2. 教练/运动科学 | `innovator` |
| D. 中国攀岩 | 按贡献归入 `elite` / `explorer` / `legend`，加 `isChineseRepresentative: true` |
| E. 文化人物（严格控制） | `legend`（≤ 总量 10%） |

## 三、各类别收录标准

### A. 竞技攀岩

**收录**：
- 奥运会攀岩奖牌得主（包括东京 2020、巴黎 2024）
- IFSC 世界锦标赛历届金牌得主
- IFSC 世界杯年度总冠军
- 打破世界纪录或完成某项目"历史首次"的运动员（如首位 onsight 9a、首位女性完攀 8c+）
- 在某个年代统治竞技排名、对后来者影响深远的运动员（即便已退役多年）

**不收录**：
- 仅参赛但未有突出成绩的运动员
- 地区/国家冠军但国际影响力有限的运动员（除非是中国运动员，见 D 类）

---

### B. 户外技术攀登

**B1 自由攀登先驱**

**收录**：
- 完成历史性首次自由攀（First Free Ascent）的攀岩者，尤其是 8c 级以上或具有里程碑意义的路线
- 推动"自由攀登"理念的先驱（如从人工攀登向自由攀登转变的推动者）
- 单次完攀等级提升对整个攀岩界有标志性意义的人

**B2 大岩壁与探险攀登**

**收录**：
- 完成了具有历史影响力的大岩壁登顶（Yosemite、Patagonia、喜马拉雅岩壁）
- 探索了此前未被攀登的重要岩壁，开辟了新的攀岩地域
- 速攀纪录创造者（特别是 El Capitan 等标志性岩壁）

**限制**：登山家（mountaineer）若其成就主要在冰雪/高海拔领域，不在本框架收录范围内。收录条件：该人物的攀岩（岩石攀爬）成就本身具有独立意义。

**B3 无保护攀登**

**收录**：
- 完成了具有历史影响力的无保护攀登，该事件在攀岩史上被广泛记录和讨论
- 不以数量取胜，以对攀岩文化的影响力为标准

**注意**：无保护攀登风险极高，收录时聚焦其对攀岩运动的影响，不美化风险行为。

---

### C. 训练与思想贡献

**C1 训练方法创始人**

**收录**：
- 发明或首创了某种具有广泛影响的攀岩训练工具或方法（如 campus board、Moon Board、系统墙训练）
- 将运动科学系统性地引入攀岩训练，改变了攀岩训练的范式

**C2 攀岩教练与运动科学先驱**

**收录**：
- 培训出了多位顶尖运动员、训练理念具有广泛影响的教练
- 出版了被广泛引用的攀岩训练科学著作，且该作品在攀岩训练圈中具有标准参考地位

**边界**：普通私教、健身房教练不在收录范围内。标准是"这个人的方法论是否影响了攀岩训练的整体走向"。

---

### D. 中国攀岩（跨类别标记）

中国攀岩人物**不单独设类**，而是根据贡献归入 `elite` / `explorer` / `legend` / `innovator`，并在数据中标记 `isChineseRepresentative: true`。

**收录与标记标准**：

| 人物类型 | 归入 category | 标记 |
|---------|--------------|------|
| 奥运会参赛 / 世界杯奖牌 / 世界纪录保持者 | `elite` | `isChineseRepresentative: true` |
| 中国本土开线先驱、户外攀岩探索者 | `explorer` | `isChineseRepresentative: true` |
| 推动中国攀岩普及的历史性人物 | `legend` | `isChineseRepresentative: true` |
| 尚未获国际奖牌但具代表性的国家队选手 | `elite` | `isChineseRepresentative: true` |

**收录门槛说明**：中国运动员与国际运动员适用**相同的收录门槛**——成就决定类别，国籍决定标记。取消"宽松标准"，改为在前端展示时通过筛选器给予同等曝光。这样做的原因是：在 `elite` 类别中与 Garnbret 并列的中国世界冠军，比被安置在独立的"中国区"更能体现其真实价值。

---

### E. 文化人物（归入 `legend`，严格控制）

原 E 类人物（导演、作家、摄影师）收录后归入 `legend` 类别。

**收录**：
- 拍摄了在攀岩圈内广为传播、具有文化影响力的纪录片（如 *Free Solo*、*The Dawn Wall* 导演）
- 出版了被广泛视为必读的经典攀岩著作

**边界**：文化人物总量不超过 `legend` 类别的 20%，整体名录占比不超过 10%。

## 四、采集重点：采访与知识输出

> 对于名人堂的每一位人物，**我们最看重的不是成就列表，而是他们输出的知识**——他们怎么思考攀岩、怎么训练、怎么克服困难。这类内容主要通过两种形式传播：文字采访和视频采访。

### 4.1 文字类采访与知识输出

**目标内容类型**（按价值排序）：

| 类型 | 说明 | 典型来源 |
|------|------|---------|
| 深度专访 | 媒体对运动员的长篇访谈，涵盖训练、理念、生涯 | 攀岩杂志、体育媒体 |
| 训练方法文章 | 运动员本人或基于其访谈整理的训练方案 | 专业训练博客、攀岩媒体 |
| 播客文字稿 | 播客节目的书面版或摘要（如有） | 播客官网 |
| 运动员自述 / 博客 | 运动员个人网站或专栏发布的文章 | 个人网站、Medium |
| 书籍 / 自传 | 运动员出版的书，尤其是涉及训练和心理的章节 | 出版物 |

**搜索关键词模式**：

```
"{运动员姓名} interview"
"{运动员姓名} training method"
"{运动员姓名} how I train"
"{运动员姓名} climbing philosophy"
"{运动员姓名} podcast transcript"
"{运动员姓名} article"
"{运动员姓名名字} 采访"
"{运动员姓名} 训练方法"
```

**记录要求**：每条文字来源需记录 URL、媒体名称、发布日期、核心话题标签（如"训练"、"心理"、"技术"、"伤病"）。

---

### 4.2 视频类采访与知识输出

**目标内容类型**（按价值排序）：

| 类型 | 说明 |
|------|------|
| 长篇深度采访 | 30 分钟以上的深度对谈，涉及训练理念和生涯经历 |
| 训练实况视频 | 展示运动员实际训练过程的 vlog 或纪录片片段 |
| 技术讲解视频 | 运动员亲自讲解某项技术或训练方法 |
| 赛前 / 赛后采访 | 简短但有洞见的赛事采访 |
| 纪录片 | 以该运动员为主角的专题纪录片 |

**需记录的视频信息**：

```
- 视频标题
- 发布平台（YouTube / Bilibili / Vimeo / Red Bull TV / 其他）
- 视频 URL
- 发布频道/媒体名称
- 发布日期（年月）
- 时长
- 核心话题标签
- 1 句话简述：这个视频的核心内容是什么
```

**主要视频平台及搜索方式**：

| 平台 | 目标内容 | 搜索建议 |
|------|---------|---------|
| YouTube | 英文深度采访、训练视频、纪录片 | `"{姓名} interview"`, `"{姓名} training"`, `"{姓名} documentary"` |
| Bilibili | 中文运动员、中文搬运/翻译视频 | `"{姓名}"`, `"攀岩 {姓名}"` |
| Vimeo | 高质量攀岩纪录片 | `"{姓名} climbing"` |
| Red Bull TV | 有赞助的运动员的专题内容 | `"{姓名} Red Bull"` |
| 运动员个人频道 | 运动员本人发布的训练 vlog | 直接搜索运动员名字的官方频道 |

**重要原则**：视频的价值在于运动员说了什么、展示了什么——**记录 URL 是最低要求，更重要的是标注核心话题**，让后续研究员能快速判断是否值得深入。

## 五、不在收录范围内

以下情况**明确排除**：

| 类型 | 排除原因 |
|------|---------|
| 纯登山家（冰雪线路为主） | 主项非岩石攀爬，属不同运动 |
| 攀树、室内休闲攀爬 | 不属于竞技/技术攀岩 |
| 健身房攀岩爱好者 / 网红 | 无竞技或技术层面的贡献 |
| 仅参赛但无突出成绩的运动员 | 不满足历史印记标准 |
| 攀岩装备品牌创始人 | 商业贡献不等同于攀岩贡献（除非本人同时是杰出攀岩者） |

## 七、优先级建议

信息收集时，建议按以下顺序推进：

| 优先级 | 新类别 | 原因 |
|--------|--------|------|
| ⭐⭐⭐ | `elite`（竞技精英） | 资料最丰富，用户关注度最高；含中国运动员 |
| ⭐⭐⭐ | `elite` 中的中国运动员 | 加 `isChineseRepresentative: true` 标记，优先完善采访内容 |
| ⭐⭐ | `explorer`（探险攀登） | 故事价值极高，是知识库（支柱一）的重要参照 |
| ⭐⭐ | `innovator`（训练革新） | 与训练手册（支柱二）关联最强 |
| ⭐ | `legend`（历史先驱） | 奠基人物，资料挖掘成本较大，但不可缺少 |
| ⭐ | `legend` 中的文化人物 | 最后补充，严格控制数量（≤ 总量 10%） |
